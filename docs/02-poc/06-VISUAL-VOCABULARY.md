# Visual Vocabulary (Nodes + Regions)

Owner: Design

## Node states
- Default: neutral fill, subtle outline
- Hover: increased outline + slight scale-up
- Selected: strong outline + persistent highlight ring
- Filter-muted: reduced alpha

## Region states
- Base region tint by cluster id
- Active region: elevated contrast and label prominence
- Inactive regions: reduced saturation

## Info hierarchy
- Primary: fragment text snippet
- Secondary: theme tags + intensity
- Tertiary: metadata (time/source)
